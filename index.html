Fantasy Map Converter

This app will create a module from the data generated by Azgaar's Fantasy Map Generator

For the best results, please follow these steps to setup a new map:

  1. Go to the Options tab and set your canvas size to 1920x1080
  2. Set the other map settings as you wish
  3. Create a new map
  4. Go to the Tools tab, click on Units, and set 1 map pixel = .96

After you have finished customizing the data, and have the map as you want it, save the .map and select it here

Now customize the visible layers, and export the map as an SVG and select it here
